# Reporting Guide

## Purpose

Reporting packages experiment results into human-readable outputs.

## Supported Reporting Outputs

- summary views
- experiment metrics
- comparison results
- DOCX exports

## DOCX Exports

Generated documents are intended for stakeholder review, audit support, and delivery packaging.

## Output Location

```text
exports/
```

## Recommended Usage

Generate reports after:

- major training runs
- comparison sessions
- compliance reviews
- replay investigations
